# District 7 Barbershop

Marketing website for **District 7 Barbershop** — *Sharp Cuts. Clean Fades. Real Craft.*

A single-page, fully responsive site with a dark, masculine aesthetic: bold Bebas Neue display type, gold accents, and real barbershop photography.

## Sections
- **Hero** — headline, stats, and booking CTA over a full-bleed shop photo
- **Services** — six-service price menu (cuts, fades, beard work, hot-towel shaves, kids' cuts)
- **Gallery** — "The Work" photo grid
- **About** — the shop's story
- **Visit** — location, hours, contact, and a booking form

## Tech
Plain HTML + CSS. No build step, no dependencies.

## Run locally
```bash
python3 -m http.server 4757
# open http://localhost:4757
```

## Structure
```
index.html      # markup
styles.css      # all styling (dark + gold theme, responsive)
images/         # barbershop photography (via Unsplash)
```
